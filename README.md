# Music Therapy Recommendation System

A comprehensive web-based application that provides personalized music therapy recommendations based on user mood, preferences, and therapeutic needs. The system features multi-language support, interactive therapy sessions, and an intuitive user interface.

This project aims to develop a system that recommends music for therapeutic purposes. It leverages data and algorithms to suggest personalized music selections to aid in well-being, mental health, and emotional healing through carefully curated therapeutic music experiences.

## âœ¨ Features

- **User Authentication**: Secure login and signup functionality
- **Mood-Based Recommendations**: Get personalized music recommendations based on your current mood
- **Multi-Language Support**: Available in English, Hindi, Kannada, and Malayalam
- **Interactive Therapy Sessions**: Guided music therapy sessions with timers and progress tracking
- **Song Management**: Browse, play, and add songs to your personal playlist
- **Feedback System**: Rate songs and provide feedback on therapy effectiveness
- **Admin Panel**: Administrative controls for managing users, songs, and system settings
- **Dark Mode**: Toggle between light and dark themes for comfortable viewing
- **Responsive Design**: Works seamlessly across different devices and screen sizes
